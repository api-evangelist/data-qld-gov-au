# Queensland Government Data (data-qld-gov-au)

Queensland Government Data is a open data portal open-data portal for Australia running **CKAN**, exposing a standardized catalog API over ~188,778 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-qld-gov-au/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-qld-gov-au-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN

## API
- **Queensland Government Data CKAN Action API** — base `https://data.qld.gov.au/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.qld.gov.au)
- Live endpoint verified: `https://data.qld.gov.au/api/3/action/package_search?rows=0` · ~188,778 datasets
